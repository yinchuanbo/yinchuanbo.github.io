实现页面刷新后不定位到之前的滚动位置

HaoTian · 2024-08-30 17:34:29

使用很简单，在页面的任意位置执行下面几行 JS 代码就可以了：

if (history.scrollRestoration) {
  history.scrollRestoration = "manual";
}
