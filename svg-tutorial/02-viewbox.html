如何使用 viewBox 属性

如何移动坐标系的原点

在 上一篇章节 中，我们定义了带有 width 和 height 属性的 SVG 元素。这些属性定义了图像在浏览器中占用的空间大小。没有其他属性时，图像的坐标系也是从这两个属性派生出来的。它从左上角的 0,0 开始，每次向下和向右增长一个像素。

0, 0
200, 200

我们可以通过设置 viewBox 属性来移动坐标系的原点。对于这个属性，我们需要设置四个数字。前两个数字是左上角的位置。如果我们将其设置为 -100,-100，原点将位于图像的中心。图像元素将根据这个新原点来定位。

-100, -100
200, 200

如何缩放图像

viewBox 属性中的最后两个数字定义了图像在图像元素视角下的内部大小。如果这些值与宽度和高度匹配，那么屏幕上的一个像素将对应于 SVG 图像中的一个单位。

如果这些值不匹配，图像元素会缩小或放大。下面，我们定义了与之前示例相同的矩形，但这次它看起来更大了。原因是，尽管图像的实际大小没有改变，但图像元素现在将可用空间视为一个 100 x 100 的网格。

0, 0
100, 100

如何用 SVG 绘制圣诞树

在下面的例子中，我们将坐标系的原点移到图像的中心。这样，我们定义的每个位置都相对于这个中心点。

-100, -200
200, 400

在这个例子中，我们将使用 polygon 元素。多边形是绘制自由形状的最简单方法。多边形有一个 points 属性，用于设置一系列通过直线连接的坐标。

我们可以通过三个三角形来制作树的树冠。从底部的三角形开始，逐层添加。

-100, -200
200, 400
-100, -200
200, 400
-100, -200
200, 400

最后，我们将树干添加为一个矩形。
