  • 我们通过 cx 和 cy 属性设置它的中心位置。
  • 我们通过 r 属性设置半径。

我们将圣诞装饰品的主体定义为一个圆形。我们将它在 x 轴上的中心位置设置为中间，值为 100，在 y 轴上的中心位置稍微向下，值为 120。我们设置了一个较大的半径，并设置了填充颜色。将鼠标悬停在元素上，查看它是如何定位的。

为了添加一些颜色，我们使用 fill 属性。这个属性类似于 CSS 中常规 HTML 元素的 background-color 属性。

颜色值可以是颜色名称，如 red，也可以是十六进制颜色代码，如 #ff0000，或者是 CSS 函数，如 rgb(255 0 0)。如果我们不想填充形状，可以将其设置为 none。
