如何在SVG中使用渐变

形状的填充可以定义为渐变。今天，我们为圣诞装饰添加一个微妙的3D效果，并制作一个雪人。

如何使用SVG绘制基本形状

升级圣诞装饰

让我们升级在 第一天 创建的圣诞装饰。下面是我们最终得到的代码：

0, 0
200, 200

我们可以通过使用渐变填充主圆形来添加一个微妙的3D效果。这里，我们在 defs 部分定义了一个 radialGradient。它的语法与CSS有所不同，但功能非常相似。

我们定义了它的ID、中心位置（通过 cx 和 cy）、设置其半径以及渐变停靠色。然后，我们可以将它作为形状的 fill 属性的值来使用。

0, 0
200, 200

制作雪人

对于雪人，我们绘制了两个圆形，并使用类似的渐变，如下所示。

-100, -100
200, 200

然后我们添加一个多边形作为鼻子，两个圆形作为眼睛，以及两条线作为手臂。简单。
