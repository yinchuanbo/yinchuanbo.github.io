如何用 SVG 绘制一只熊

让我们绘制一只熊，以练习绘制 三次 和 二次贝塞尔 曲线。

我们从耳朵开始。这是两个圆形，它们都设置了 fill 和 stroke 属性。我们还在 SVG 元素上定义了一个类来设置背景颜色。请注意，这里我们设置的是常规的 background-color CSS 属性，而不是 fill 属性。填充属性适用于形状，但 SVG 元素本身并不是一个形状。

-100, -100
200, 200

接下来，让我们添加脸部和眼睛。这是我们之前没有涉及的另一件事。早些时候我们看到可以使用 rx 属性来圆角矩形。我们可以更具体地设置 Y 轴上的不同半径，使用 ry 属性。如果没有设置 ry，它将回退到 rx 的值。

-100, -100
200, 200

接下来，让我们绘制鼻子。我们使用 path 元素来实现这一点。

我们将其上部绘制为一个 三次贝塞尔曲线，而两个底角则使用两个 二次贝塞尔曲线。将鼠标悬停在代码或图像中的坐标上，以查看它们的位置。

-100, -100
200, 200

最后，我们可以将鼻子和嘴巴添加为两个路径。鼻子使用一个 三次贝塞尔曲线，而嘴巴则使用两个 二次贝塞尔曲线。

-100, -100
200, 200

一旦我们将所有部分组合在一起，就得到了左侧的图像。
