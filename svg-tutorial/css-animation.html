如何使用 CSS 为 SVG 添加动画

你可以将一些SVG属性移到CSS中，甚至使用CSS关键帧对它们进行动画处理。在这个示例中，我们创建了一个旋转的风车。

绘制风车的头部

我们从组合风车的头部开始。由于它的中心对称性，我们可以定义一个单独的臂，并通过旋转重复这个元素。

完成风车并添加CSS动画

为了完成风车，我们添加了一个支架，并将风车的头部包裹在两个组元素中。外部组用于定位头部，内部组负责旋转。

我们将外部组通过 translate 移动-50单位，以便定位头部。

我们将id windmill-head 分配给内部组，并在CSS中设置一个关键帧动画来旋转头部。
