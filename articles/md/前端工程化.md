## Vue CLI 脚手架

### 安装 Vue CLI

```sh
npm install -g @vue/cli
```

使用 vue 命令创建项目

```sh
vue create 项目的名称
```

### Vue CLI 新建项目

**注意：** Vue3 官网推荐 Volar 插件

**第一步：使用 Vue CLI 的 vue 命令新建一个名为 “01_vuecli_demo” 的 Vue3 项目**

```sh
vue create 01_vuecli_demo
```
